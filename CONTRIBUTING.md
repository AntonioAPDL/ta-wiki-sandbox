# Contributing

This repo is a teaching sandbox. Keep changes small, focused, and easy to review.

## Workflow (forks for public repos)
- Fork the repo on GitHub.
- Push your branch to your fork.
- Open a PR from your fork to `AntonioAPDL/ta-wiki-sandbox`.

## Branch naming
- Use `username/<short-topic>`.
- Examples: `antonio/fix-typo`, `lee/link-fix`.

## Commit messages
- Use `docs: ...` for documentation changes.
- Use `fix: ...` for small corrections (typos, links).

## PR scope
- Single purpose, small diff.

## Quality checklist (required)
- Preview Markdown in VS Code.
- Check links you touched.
- Ensure the commit message is clear and scoped.
